# 株式会社縁 ホームページサンプル v1

北九州市八幡東区のリフォーム会社「株式会社縁」のホームページサンプル。

## 構成

- `index.html` — トップページ
- `about.html` — 会社概要
- `services.html` — 事業内容
- `works.html` — 施工事例
- `contact.html` — お問い合わせ
- `assets/css/style.css` — 共通スタイル
- `assets/js/main.js` — 共通スクリプト

## プレビュー

GitHub Pages: https://mappy0509.github.io/enishi-website-sample/

## ローカル確認

```bash
python3 -m http.server 8080
# → http://localhost:8080
```

## 注意事項

- 本リポジトリはサンプルのため、画像はUnsplashの仮素材を使用
- 施工事例・代表写真などは仮データ
- フォーム送信は動作しません（サンプルのため）
